🚀 How to Test

  1. Install the Extension: Load the unpacked extension in Chrome
  2. Complete Setup: Add your Gemini API key and upload photos
  3. Test Methods:
    • Select Item: Click extension icon → "Select Item" → Click any clothing image below
    • Screenshot: Click extension icon → "Screenshot" to capture this entire page
    • Image URL: Right-click any image → Copy image address → Use in extension
  4. Keyboard Shortcut: Press Ctrl+Shift+T to start selection mode
